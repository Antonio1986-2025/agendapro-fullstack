========================================
🔐 CONFIGURAÇÃO RÁPIDA - GITHUB SECRETS
========================================

COPIE OS VALORES ABAIXO E ADICIONE COMO SECRETS NO GITHUB:
(Settings → Secrets and variables → Actions → New repository secret)

========================================
1. VPS_SSH_KEY
========================================
Execute na sua máquina:
ssh-keygen -t ed25519 -C "github-deploy" -f ~/.ssh/vps_deploy
cat ~/.ssh/vps_deploy

↓ COPIE TODO O CONTEÚDO (incluindo BEGIN e END) ↓

Depois adicione a chave pública na VPS:
cat ~/.ssh/vps_deploy.pub
ssh seu-usuario@seu-ip-vps
nano ~/.ssh/authorized_keys
# Cole a chave pública aqui

========================================
2. VPS_HOST
========================================
O IP ou domínio da sua VPS

Exemplo:
123.45.67.89

========================================
3. VPS_USER
========================================
Usuário SSH da VPS

Exemplo:
root
ou: ubuntu

========================================
4. VPS_PATH
========================================
Caminho COMPLETO do projeto na VPS

Para descobrir, execute na VPS:
ssh seu-usuario@seu-ip-vps
cd /caminho/do/projeto
pwd

Exemplo:
/root/agendapro-fullstack
ou: /home/ubuntu/agendapro
